# Portfolio

Portfólio pessoal de desenvolvedor web built with React + TailwindCSS.

## Como usar

1. Instale as dependências: `npm install`
2. Rode o servidor: `npm run dev`
3. Abra http://localhost:5173

## Personalização

- Substitua `src/assets/docs/curriculo.pdf` pelo seu currículo
- Edite os arquivos em `src/constants/` com seus dados
